Curriculum Mashup

Last year the world has seen the rise of many new platforms providing free courses on the web. Based on my personal experience the quality is very high, but they are still a bit scattered. The student has to figure out by himself what courses might be useful, and no-one's offering a full curriculum yet. 

This issue could be easily solved by making a curriculum mashup: a website combining the courses offered by separate organizations to meaningful packages. Curriculum to become a Web Entrepreneur might look something like this:

Intro to Computer Science (Udacity) http://www.udacity.com/view#Course/cs101/CourseRev/apr2012/Unit/671001/Nugget/675002
Web Fundamentals (CodeAcademy) http://www.codecademy.com/tracks/web
Web Application Engineering (Udacity) http://www.udacity.com/view#Course/cs253/CourseRev/apr2012
Human Computer Interaction (Coursera) https://class.coursera.org/hci/class/index
Creative, serious and playful science of Android Apps (Coursera) https://www.coursera.org/course/androidapps101
Grow to greatness: Smart growth for private businesses (Coursera) https://www.coursera.org/course/growtogreatness

During these courses the student would learn the basics of programming web applications, learn about user needs, build and test prototypes and turn the prototype to a real application. There should also be some more courses on how to run the business, but for now I only managed to find one on growth.

  • Idea is to make a mashup of already existing free web courses from Coursera, Udacity, etc. to form a short curriculum which gives the student necessary skills to develop a web application and start running a small business
  • Content
    • HCI course to find a relevant problem and design an elegant solution
    • Code Acadamy: html and CSS
    • Udacity: Python basics and Web application engineering
    • OpenIDEO to give a context or topic
    • Kickstarter for funding
  • After taking the basic courses on technical skills, the students would participate to HCI course, the topic of which comes from OpenIDEO challenge
  • Benefits: 
    • Integrates already existing but a bit dispersed courses from different platforms to a neat package
    • More participants to OpenIDEO challenges. They are motivated by personal learning, but help solve some real problems. They also learn the necessary design skills as they go
    • The end result would be a big bunch of prototype web applications, some of which might even become a real business, with the help of Kickstarter or other crowdfunding solutions
  • Needed resources: Not much. Already build a prototype (This could be done on Weebly)
  • Could also be turned to a game, where each skill set is like levels: You're a level 1 programmer!
  • Companies could make them the requirement: "We're looking for someone, who can do these <list of courses>"
