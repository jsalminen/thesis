How To... Create a link in your post

Meena Kadri

posted this on October 11, 2011 02:55

 

It's often really helpful to add a link in your post so that others can check out more detail about what you're discussing.

You can do this using the Formatting Bar above the Description field when you submit an Inspiration or Concept

– which you can see pictured in the image above



Let's say you're talking about this article on Local Food: http://www.fastcompany.com/1785988/emily-pillotons-studio-h-high-sc...

You can have it link directly to your text like this: Check out more on Local Food rather than posting the long link.

It's less interruptive to reading and more likely to attract clicks than long links. And it only takes a few seconds!

1. Simply highlight the text you want to link (in this case Check out more on Local Food)

2. Hit the link button pictured above (a chain link with a '+' sign next to it)

3. You will then be prompted to add the link

4. Hit Enter and you're done – text all linked & easy for folks to access!
